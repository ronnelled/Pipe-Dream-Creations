# Pipe-Dream-Creations
This repository contains the development of a website for Pipe Dream Creations, a small pipe cleaner craft business specializing in handmade decorations, custom designs, and event pieces. The purpose of this project is to design a functional business website that showcases products, communicates brand identity, and enables customer engagement.
The MVP includes:
Homepage
Navigation
About page
Product gallery
Contact form
Responsive design
Basic branding
Footer
