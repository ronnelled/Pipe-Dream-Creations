# Project Charter
## Pipe Cleaner Business Website

### Problem
The pipe cleaner craft business currently lacks an online presence, limiting product visibility and customer engagement.

### Users
- Parents
- Teachers
- Event planners
- DIY craft enthusiasts
- Local customers

### Success Metrics
- Website deployed publicly
- 4 core pages completed
- Responsive design
- Functional contact form
- 10+ closed GitHub issues

### Scope

#### In Scope (MVP)
- Home page
- About page
- Product gallery
- Contact form
- Responsive layout

#### Out of Scope
- E-commerce checkout
- Payment processing
- User login accounts
- Inventory system
