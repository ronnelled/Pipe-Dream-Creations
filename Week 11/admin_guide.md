# Admin Guide

## Updating Content
- Edit `index.html` for homepage content
- Edit files in `/pages` for secondary pages
- Update images and text as needed
- Commit changes to a feature branch
- Open a pull request using the repository template

## Pre-Release Checklist
- All pages load
- No placeholder text remains
- No console errors
- Links work
- Images display correctly
- Workflow passes in GitHub Actions

## Deployment
Changes merged into `main` trigger the GitHub Actions workflow and deploy to GitHub Pages.
