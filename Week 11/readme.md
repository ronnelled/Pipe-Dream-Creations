Pipe Dream Creations is a small business website for showcasing handmade pipe cleaner decorations, custom gifts, and event pieces.

## Project Structure
- `index.html` - Homepage
- `pages/about.html` - About page
- `pages/products.html` - Product gallery page
- `pages/contact.html` - Contact page
- `pages/faq.html` - FAQ page
- `.github/workflows/main.yml` - CI/CD and Pages deployment
- `.github/pull_request_template.md` - Pull request checklist

## How to Run Locally
1. Download or clone the repository.
2. Open `index.html` in a browser.
3. Test navigation to each page.
4. Confirm images, links, and layout load correctly.

## Testing Checklist
- Navigation works on all pages
- No broken links
- No placeholder text
- No console errors
- Layout works on desktop and mobile view
- Images display correctly
- Accessibility basics checked:
  - skip link
  - alt text
  - heading order
  - keyboard navigation

## Deployment
This project is deployed with GitHub Pages using GitHub Actions.
