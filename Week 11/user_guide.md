# User Guide

## Purpose
This website helps visitors learn about Pipe Dream Creations and browse handmade products.

## How to Use the Site
1. Open the homepage.
2. Use the navigation menu to visit About, Products, Contact, and FAQ.
3. Browse featured items and product information.
4. Use the Contact page to learn how to reach the business.

## Expected User Experience
- Simple navigation
- Clear product information
- Mobile-friendly layout
- Accessible images and headings
