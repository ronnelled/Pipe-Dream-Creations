# Pipe Dream Website – Usability Test Plan

## Goal

Evaluate whether users can easily navigate the Pipe Dream website and find products.

## Participants

5 users (students or online shoppers).

## Test Tasks

Task 1
Find a pipe cleaner product you would purchase.

Task 2
Use the category filters to narrow product results.

Task 3
Open a product image using the lightbox feature.

Task 4
Find the FAQ section and locate a return policy.

Task 5
Sign up for the newsletter.

## Metrics

* Time to complete each task
* Errors made
* User satisfaction (1–5 rating)

## Questions After Testing

1. Was the navigation easy to understand?
2. Could you find products quickly?
3. Did any page feel confusing?
4. What improvements would you suggest?

## Success Criteria

* 80% of users complete all tasks
* Average satisfaction rating ≥ 4/5
